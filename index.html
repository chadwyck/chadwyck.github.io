    Open-Source Activiti UI Builder

    Last summer I had the chance to work on an awesome project with BP3 Global in Austin, Texas creating an easy-to-use UI builder for Business Process Management user tasks running in the open-source Activiti BPM engine. The result was beyond any of our expectations at the beginning of the summer. Running as a Node.js web application with a customizable, responsive, Bootstrap-based interface, the UI builder gave an out-of-the-box webserver to run tasks and a drag-and-drop interface to create UI's for tasks. Check out BP3's post about the project here.
